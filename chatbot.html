AI Chatbot with ChatGPT

Welcome to the Chat Project! This is a Django chatbot. The chatbot allows users to interact with an AI assistant using the OpenAI API, asking questions and receiving responses in a conversational manner.

How it Works:

When a user interacts with the chatbot, they are presented with a chat interface where they can enter their queries. The chatbot processes the user's input and generates a response using the OpenAI API. The response is then displayed on the screen, simulating a conversation with the AI assistant.
The chatbot's behavior is influenced by the selected personality or the custom personality created by the user. Users can customize the behavior and adjust the temperature, controlling the randomness of the chatbot's responses. The chatbot utilizes advanced natural language processing techniques to understand and generate meaningful responses based on the given query and context.
The project also provides features for managing user profiles, allowing users to update their personal information such as name, email, and password. The chat history page displays the first entry for each unique personality, providing a glimpse of past conversations.
