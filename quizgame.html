Quiz Game

QUIZ GAME is a custom quiz game. The application allows users to create, share, and play quizzes with questions from multiple-choice and true or false formats. It was designed to offer an interactive and engaging experience, providing entertainment and learning opportunities for users.
It's a Web application (Full Stack), made using Python (Django), Javascript, HTML and CSS.

Features and Functionalities:
1. Creating Quizzes:
- Users can create their own custom quizzes by adding a title, description, and cover image.
- They can add questions with multiple-choice or true or false options.
- Users have the option to set a time limit for answering the questions.

2. Interactive Gameplay:
- Quizzes can be shared with other users, allowing them to play and test their knowledge.
- During the game, users' answers are evaluated automatically, and the results are displayed in real-time.
- Players can see their score and rank against other participants.

3. Personalization:
- Users can add tags to their quizzes, making them easier to be found by other players interested in specific topics.

4. Quizzes Management:
- Users can view and manage all quizzes created by them.
- They can edit, delete, or mark quizzes as private, restricting access only to selected users.

How The Game Is Played

One Host, Multiple Players

One Host presents and controls the game. The Players log into the specific game and choose the answers in a pre determined time period. When all quizzes are done, the player with the most points wins.

Landing Screen

Landing Screen

When going to the website, the user lands at this page. Here it should be specified if the user is a PLAYER or a game HOST.

Host Screens

Log In or Create Account

When the user selects the option HOST, he/she is sent to this page, that prompts him to log in or create an account.

Create Account

This page lets the user input all his information for creating an account. It has all the normal SECURITY CHECKS such as asking for the password twice and checking the database for any users with the same data.

Login Page

Page for the user to log in. It also features MESSAGES to give the user some feedback while trying to log in.

Main Host Page

Here is where most of the HOST's features are at. The host can Create, delete, search and start quizzes. The existing quizzes show here, with pagination support for when there are too many.

Create New Quiz

From the main host page, the host can select the CREATE QUIZ button and fill in all the information for his new quiz game.

Create Questions: Quiz || True or False

This is the place the questions are filled. The host will choose if it's a Quiz OR True or False question, how many seconds the players will have to answer, type the questions and select an image. The answers must be typed in the colored boxes, and the correct answer must be ticked with the selector icon. Quiz questions have 4 answer options and True or False questions have 2 answer options.

Edit Question

The host also has the ability to edit any information from any question that he wishes. Just select the question from the left menu, alter the data and save.

Edit Quiz Info

It's also possible to edit the information from the Quiz itself, using the CONFIGURATION button.

Host Lobby

When the host selects START GAME from any quiz, he/she will be sent to this LOBBY page which has the information for the players to see, such as the game PIN and the players that already logged into the game. When everybody is logged in, the host selects the START GAME button, and the game starts simultaneously for every player.

Question and Countdown

The question will show up on the screen, as well as the countdown for the players to answer. Each player answers on his/her own device.

Partial Score

After the time for each question is over, the host will show a scoreboard for everyone to see where they stand in terms of points.

Final Score

When all questions are finalized, the final score is shown at the host screen congratulating the winner and showing the score of all players.

Statistics: Questions and Answers

The first statistics shown are all the questions and answers, and also its hit rate among the players.

Statistics: Game Stats

The second statistics button indicates the total hit rate of players among all questions.

Statistics: Players

This third statistics button shows each player's scoreboard, along with the total score and hit rate.

Played Quizzes History

This page lets the host see all the quizzes that were played. It's possible to search played quizzes, check the scoreboard of each one, as well as deleting it or extracting the information to excel.

Edit Host Profile

Here the host can edit his personal information, change password and backup the hole database.

Player Screens

Player Info

This is the first screen that the user is shown after selecting that he/she is a PLAYER, instead of a HOST. Here the player inputs all his information, as well as the game PIN that is shown on the HOST screen.

Player Lobby

After inserting the information, the player is sent to the PLAYER LOBBY, where he will wait until the HOST decides to start the game.

Answers: Quiz

The player will be shown 4 options to choose from when the game is a QUIZ game, as well as a countdown for the remaining answering time.

Answers: True or False

The player will be shown 2 options to choose from when the game is a TRUE OR FALSE game, as well as a countdown for the remaining answering time.

Answer Selected

After selecting an answer, the other choices will not be shown again, and the choice can not be undone.

Partial Score: Not Top 3

The partial score is shown after each countdown ends. If the player is not among the top 3, he/she will see its position at 4th line of the scoreboard with the current position.

Partial Score: Top 3

If the player is among the top 3, only the top 3 players will show up at the scoreboard.

Final Player Position

Each player is shown his/her own final position.
